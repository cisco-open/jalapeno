get /agent/config
(getAgentConfig)
Get the global config for this mu agent

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Return type

AgentConfig

Example data

Content-Type: application/json
{
  "agentAPI" : {
    "url" : "aeiou"
  },
  "agentManager" : {
    "heartbeatInterval" : "aeiou",
    "ctrlAPIEndpoint" : "aeiou"
  },
  "agentRunner" : {
    "enable_debug" : true,
    "log_level" : "aeiou",
    "datacenter" : "aeiou",
    "clientConfig" : {
      "servers" : [ "aeiou" ],
      "no_host_uuid" : true,
      "enabled" : true
    },
    "region" : "aeiou",
    "serverConfig" : {
      "bootstrap" : 123,
      "enabled" : true
    },
    "advertise" : {
      "key" : "{}"
    },
    "data_dir" : "aeiou"
  }
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Current config of the agent AgentConfig
